# main-iqy-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容涉及日常记录、技术笔记、工具页面等，以静态 HTML 文件的形式直接呈现。仓库不针对任何特定网站或域名，仅作为个人整理的页面集合存档。

## 目录说明

```
├── html/          # 存放所有 HTML 页面文件
├── assets/        # 页面引用的资源文件（如 CSS、JavaScript、图片等）
├── docs/          # 其他辅助文档或说明
└── README.md      # 本文件
```

- `html/` 目录下的每个 HTML 文件均为一个独立的页面，可直接在浏览器中打开。
- `assets/` 目录按页面或功能进行子目录划分，方便资源管理。
- `docs/` 目录可存放与页面相关的补充说明、更新日志等。

## 页面归档说明

- 每个 HTML 页面均为独立、完整的文件，通常包含内联或引用的样式和脚本。
- 页面内容主要来源于个人日常记录、学习笔记或临时工具需求，未经过系统性整理。
- 归档时尽量保持页面原始结构和功能，不进行额外修改或优化。
- 部分页面可能依赖外部资源（如字体、图标库等），若无法加载可能影响显示效果。

## 维护说明

- 本仓库不定期更新，新增页面或修正已有页面。
- 不承诺对页面内容进行持续维护或更新。
- 若发现页面显示异常或链接失效，欢迎通过 Issues 反馈，但修复时间不定。
- 仓库仅供个人归档和学习参考使用，不提供任何形式的服务或支持。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `html/` 目录下的任意 HTML 文件。
3. 也可通过 GitHub Pages 等静态托管服务发布后在线访问（需自行配置）。
